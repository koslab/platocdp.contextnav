platocdp.contextnav Installation
--------------------------------

To install platocdp.contextnav using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``platocdp.contextnav`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        platocdp.contextnav

* Re-run buildout, e.g. with:

    $ ./bin/buildout
